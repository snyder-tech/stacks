# STX Docs

STX keeps the docs site thin.

It consumes the BDX docs library stack instead of reimplementing a parallel
docs engine in this repo.

## What lives here

- a small Analog app in `apps/docs-site`
- docs rendering and navigation from BDX libraries
- repo-level guidance on the STX and BDX boundary
