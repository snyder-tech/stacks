# STX Knowledge Base

`@snyder-tech/stx-knowledge-base` is the first stack package in this repo.

It stays intentionally small for now:

- a typed stack definition API
- sensible defaults for an Analog-powered docs or knowledge-base app
- a package boundary that the release and CI pipeline can publish and verify

The docs site consumes BDX docs libraries so this stack can focus on the
solution layer instead of framework plumbing.
