# STX Branding

## One-liner

Complete open-source stacks for the AI-native web and beyond. By Snyder Tech.

## Positioning

- `BDX` is the tools and package layer
- `STX` is the complete solution-stack layer
- the docs site is a thin consumer of BDX docs libraries

## npm Prefix

- `@snyder-tech/stx-*`

Examples:

- `@snyder-tech/stx-knowledge-base`
- `@snyder-tech/stx-analog-ai-chat`
- `@snyder-tech/stx-llm-rag-stack`
