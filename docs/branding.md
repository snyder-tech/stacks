# STX Branding

## One-liner

Complete open-source stacks for the AI-native web — AnalogJS knowledge bases
and beyond. By Snyder Tech.

## Positioning

- `BDX` is the tools and package layer
- `STX` is the complete solution-stack layer

## npm Prefix

- `@snyder-tech/stx-*`

Examples:

- `@snyder-tech/stx-knowledge-base`
- `@snyder-tech/stx-analog-ai-chat`
- `@snyder-tech/stx-llm-rag-stack`
